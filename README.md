# Sales Performance Data Analysis

## Objective
To analyze sales data and derive actionable business insights using SQL.

## Tools Used
- SQL
- Microsoft Excel (for initial data inspection)

## Business Questions Answered
- What is the total revenue?
- Which products generate the most sales?
- How does revenue change monthly?
- Which cities contribute the most?
- Who are repeat customers?

## Key Insights
- Electronics products contribute the highest revenue.
- Metro cities generate the majority of sales.
- A small group of customers place multiple orders.
- Revenue shows variation across months.

## Conclusion
This project demonstrates fundamental data analytics skills including querying data, identifying trends, and generating business insights.
